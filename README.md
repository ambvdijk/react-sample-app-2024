# react-sample-app-2024

This is a basic sample of a 2024 React Web Application without Babel using only
1. React
2. TypeScript
3. Vite
4. Jest (using ts-jest)


To install all dependencies:
```shell
> npm install
```

To start the project:
```shell
> npm run start
```

Run tests:
```shell
> npm run test
```
